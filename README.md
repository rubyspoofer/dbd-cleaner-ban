# dbd-cleaner-ban ПОДХОДИТ ТОЛЬКО ДЛЯ MICROSOFT STORE
Клинер для игры Dead by Daylight который поможет обойти бан по железу / бан по лицензии от Microsoft.

**Системные требования:**

ОС: Windows 10/11

Платформа: Microsoft Store (Xbox)

## ВАЖНО
Важно: для полного выхода из забаненого акка МС вас нужно найти в Настройки > Учётные записи > Ваши данные > Отменить автоматический вход во все приложения Майкрософт (сделай учётку локальной)

Далее: Настройки > Учётные записи > Электронная почта и учётные записи > Выберите вашу вбаненную учётку и удалите её.
